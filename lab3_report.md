# Лабораторная работа №3: Запуск бота для реального использования
## Описание деплоя

**Какой способ?**
Локальный запуск с доступом через ngrok (Вариант 1).

**Почему именно этот?**
- Не требует платного хостинга
- Подходит для тестирования
- Быстрый старт без регистрации на новых платформах
- Мой компьютер может быть включен на время тестирования

**Видео работы бота:** https://drive.google.com/file/d/1L8ywLmdkRZkkYOCbYCXBg3J3mLRhzimI/view?usp=sharing

**Результат:**
- Бот отвечает корректно
- Кнопки работают удобно
- Экскурсии получаются уникальные, так как разный набор картин и настройки
- Содержание о картинах можно в дальнейшем дополнять и менять.
Также в дальнейшем хочу научиться делать постоянный деплой 24/7.
