Создай Telegram-бота на Python с использованием библиотеки python-telegram-bot.

Функционал бота:
- Бот-гид по картинной галерее (Русский музей, СПб).
- При команде /start бот приветствует пользователя и предлагает начать экскурсию (Вопрос от бота "Начать экскурсию?" с кнопкой “Да!”).
- После начала бот спрашивает: "Какую экскурсию вы хотите?" (две кнопки: "Короткая" и "Подробная").
- Затем бот спрашивает: "Что вас интересует больше всего?" (4 кнопки: "История", "Сюжет", "Биография", "Живопись").
- После выбора бот формирует экскурсию из 3 случайных картин. Для каждой картины он выдает:
  - Название картины и имя фамилию художника, год написания картины.
  - Текст-заглушку (пока шаблонный текст, например: "«Последний день Помпеи», К. П. Брюллов (1833) — вот тут будет описание про [выбранный интерес]")
- В конце экскурсии бот просит оценить её от 1 до 5 (просто отправить цифру), после чего предлагает начать новую экскурсию (аналогично тому, что было в начале: Вопрос от бота "Начать экскурсию?" с кнопкой “Да!”)
- Данные о картинах (10 штук) хранить временно в списке или словаре внутри кода (пока без CSV). Картины и художников должны быть реально существующие из русского музея, но тексты описаний — заглушки.

Требования:
- Кнопки (ReplyKeyboardMarkup или InlineKeyboardMarkup)
- Состояния (ConversationHandler) для пошагового опроса
- Хранение выбора пользователя (длина экскурсии и интерес) в памяти в процессе диалога
- Бот должен быть простым и понятным
- Код должен быть хорошо прокомментирован
- Использовать файл для хранения данных (JSON или SQLite)
- Добавить обработку ошибок

Создай:
1. Файл bot.py с кодом бота
2. Файл requirements.txt с зависимостями
3. Файл README.md с инструкцией по запуску
4. Файл .env.example для примера конфигурации
