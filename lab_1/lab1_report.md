# Лабораторная работа №1: Создание Telegram-бота без программирования

## Описание задачи

**Какую проблему решает бот?**
Бот помогает пользователям знакомиться с искусством в интерактивном формате. Вместо скучного чтения статей пользователь получает персонализированную экскурсию по картинной галерее.

**Почему выбрали эту задачу?**
Мне очень интересна тема искусства. При посещении картинной галереи хочется не просто смотреть на картины, а изучасть понимать контекст. Бот-гид делает экскурсию увлекательной, адаптируя контент под интересы пользователя (история, сюжет, биография, живопись).

## Стек технологий

| Библиотека | Зачем |
|------------|-------|
| python-telegram-bot v20+ | Работа с Telegram API, ConversationHandler, кнопки |
| random (встроенная) | Случайный выбор картин |
