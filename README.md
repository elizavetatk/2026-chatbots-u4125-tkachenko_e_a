# Репозиторий для выполнения заданий курса Vibe Coding: AI-боты для бизнеса

### Описание учебной программы курса
Этот курс научит вас создавать Telegram-ботов для решения бизнес-задач, используя AI-инструменты вместо написания кода вручную. Вы узнаете:
- Как использовать LLM для генерации кода
- Как создавать ботов с нуля без программирования
- Как подключать ботов к данным и API
- Как деплоить ботов и собирать обратную связь
- Как улучшать продукты на основе фидбека пользователей
